The Lang Backstory

Lang was not initially created to be a fully-fledged language learning application. Lang was originally a python script that I ran in my terminal to study for a french test.

When it started out, Lang was very bare-bones. It only incuded the built in French verbs- the ones that me and my friends needed for our current unit.

I ended up realizing that I liked using Lang more than other studying options, because if there were any issues or things i wanted to be added, i could just do it myself. If Lang were to continue being used, it needed a simpler way to add new verbs. This is when I developed Lang Studysheets.

Lang can continue being free, but also provide a full user-created Studysheet system with a clever implementation of localstorage and filereader, which work together to emulate a server experience for free.
